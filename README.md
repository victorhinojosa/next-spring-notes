## API Endpoints
- `GET /api/notes`: Retrieve all notes
- `PUT /api/notes`: Create a new note
- [WIP] `PATCH /api/notes/{id}`: Update a note
- [WIP] `DELETE /api/notes/{id}`: Delete a note
